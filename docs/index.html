GitHub Pages Remote Console

Codex Remote Command Portal

이 페이지는 무료 정적 페이지입니다. 실제 명령 전송은 GitHub Issue로 기록되고, 로컬 PC의 poller와 executor가 그 이슈를 읽어 처리합니다.

Repository frysquirrel-A/codex-local-agent-tools
Command Channel GitHub Issues with remote-command label
Execution Rule v1 auto-exec only runs structured JSON commands from the trusted owner account

원격 명령 보내기

제출하면 GitHub 이슈 작성 화면으로 이동합니다. v1 자동 실행을 원하면 Command 칸에 JSON을 넣어야 합니다.

구조화 명령 예시

아래처럼 JSON 한 줄을 Command 칸에 넣으면 로컬 executor가 deterministic 하게 처리합니다.

Browser

{"mode":"browser-command","action":"navigate","url":"https://example.com"}

LLM

{"mode":"llm-prompt","provider":"chatgpt","prompt":"Summarize the latest issue.","send":true}

Desktop

{"mode":"desktop-command","action":"screen-size"}

열려 있는 원격 명령

공개 저장소 기준으로 최신 open issue를 보여줍니다.

운영 방식

  • 이 페이지는 무료 정적 호스팅이고 서버 코드는 없습니다.
  • 실제 메시지 전달은 GitHub Issue를 사용합니다.
  • 로컬 PC에서는 poller가 새 이슈를 큐로 가져오고 executor가 safe item만 처리합니다.
  • 금융 실행, 과금, 파괴적 작업은 자동 실행 대상에서 제외됩니다.
